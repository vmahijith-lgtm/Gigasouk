Upload your production logo files in this folder.

Recommended files:
- `logo.svg` (preferred scalable logo; currently a placeholder)
- `logo.png` (optional raster fallback)
- `favicon.ico` (optional; if missing, app uses `logo.svg` as tab icon)

Public URL path examples:
- `/brand/logo.svg`
- `/brand/logo.png`
- `/brand/favicon.ico`
